2. Vector Fields

Vector Fields

Vector Fields

A vector field is a function \[ \vec{F} = (F_1, \dots, F_n) : \RR^n \to \RR^n \]

Vector Fields Examples

  • Position Vector: \(\vec{F}(x, y, z) = (x, y, z) = r\)
  • Rotation Field: \(\vec{F}(x, y) = (-y, x)\)
  • Inverse Square Law: \(\vec{F}(r) = \frac{C}{\abs{r}^2} \frac{r}{\abs{r}}\)

3. Gradient Fields

Gradient Fields

Gradient Fields

A vector field of the form \(\vec{F}(r) = \nabla f (r)\) is called a gradient vector field.

Here \(r = (x_1, \dots, x_n)\)

Gradient Fields Examples

  • \(f(r) = \frac{\abs{r}^2}{2}\)
  • \(f(r) = x^2 y^2\)

Uniqueness of Gradient Fields

\(\nabla f = \nabla g\) if and only if \(g(r) = f(r) + C\).

Level Sets

Let \(f\) be a function with \(\nabla f \neq 0\). Then \(\nabla f\) is perpendicular to the level sets of \(f\).
