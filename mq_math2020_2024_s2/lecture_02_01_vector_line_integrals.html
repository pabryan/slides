2. Scalar Line Integrals

Scalar Line Integrals

Scalar Line Integrals

The integral of \(f\) along \(C\) is \[ \int_C f ds = \int_a^b f(\vecna{c}(t)) \abs{\vecna{c}'(t)} dt \]

Scalar Line Examples

\[ \vecna{c}(t) = (\cos t, \sin t, t) \] \[ f(x, y, z) = x^2 + y^2 + z^2 \]

3. Vector Line Integrals

Vector Line Integrals

Work

\[ W = F \cdot V \]

Work Along a Curve

\[ W = \int_C \vec{F} \cdot \vec{T} ds = \int_a^b \vec{F}(\vecna{c}(t)) \cdot \vecna{c}'(t) dt \]

\[ ds = \abs{\vecna{c}'}dt \quad \vec{T} = \frac{\vecna{c}'}{\abs{\vecna{c}'}} \]

Example

Calculate \(\int_C \vec{F} \cdot \vec{T} ds\) where

\[ \vec{F} (x, y) = (xy, x + y^2) \quad \vecna{c}(t) = (t, t^2), 0 \leq t \leq 1 \]

2d Notation

For \(\vec{F} = (P, Q)\) and \(c(t) = (x(t), y(t))\)

\begin{align*} \int_{C} \vec{F} \cdot \vec{T} ds &= \int_a^b P \frac{dx}{dt} dt + Q \frac{dy}{dt} dt \\ &= \int_C P dx + Q dy. \end{align*}

3d Notation

For \(\vec{F} = (P, Q, R)\) and \(c = (x(t), y(t), z(t))\)

\begin{align*} \int_{C} \vec{F} \cdot \vec{T} ds &= \int_a^b P \frac{dx}{dt} dt + Q \frac{dy}{dt} dt + R \frac{dz}{dt} dt \\ &= \int_C P dx + Q dy + Rdz. \end{align*}

Example

\[ \int_{C} x dx + y dy + z dz \] along \(\vecna{c}(t) = (\sin t, \cos t, t)\), \(0 \leq t \leq 2\pi\).

4. FTC for Gradients

FTC for Gradients

FTC for Gradients

\[ \int_{C} \nabla f \cdot \vec{T} ds = f(\vecna{c}(b)) - f(\vecna{c}(a)) \]

Example

  • \(\vecna{c}(t) = \big(t^2/4, \sin^3(t\pi/2)\big)\) \(t \in [0, 1]\)
  • \(f(x, y) = xy\)

\[ \int_{C} ydx + xdy = \frac{1}{4} \]

Conservative Vector Fields

A vector field is conservative if the work done around a closed loop is zero.

Conservative Vector Fields

The following conditions are equivalent:

  1. \(F\) is conservative
  2. \(F = \nabla f\) for some scalar field \(f\)
  3. The work done by \(F\) along a path joining points \(p\) and \(q\) is independent of the path taken

Inverse square law

\[ \vec{F}(x, y) = \frac{1}{(x^2 + y^2)^{3/2}} (x, y), \quad (x, y) \ne (0, 0) \]

\[ \vec{F} = \nabla \left(\frac{-1}{\sqrt{x^2 + y^2}}\right) \]
