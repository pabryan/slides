2. Divergence Theorem

Divergence Theorem

Divergence Theorem

Let \(\Omega \subseteq \RR^3\) be a connected open set with boundary surface \(S = \partial \Omega\). For any vector field \(F\), \[ \iiint_{\Omega} \divg \vec{F} dV = \int_{S} \vec{F} \cdot \vec{N} dA \] where \(\vec{N}\) is the outward unit normal to \(S\).

Divergence Theorem

  • Total divergence equals flux through boundary

    The amount of material leaving a region is the amount passing through the boundary!

3. Examples

Examples

  • \(\vec{F} = 2x\vec{e}_1 + y^2 \vec{e}_2 + z^2 \vec{e}_3\)
  • \(\mathbb{S}^2 = \{x^2 + y^2 + z^2 = 1\}\)
  • \(\mathbb{B}^3 = \{x^2 + y^2 + z^2 < 1\}\)

\[ \iint_{\mathbb{S}^2} \vec{F} \cdot \vec{N} dA = \iiint_{\mathbb{B}^3} \divg \vec{F} dV = \frac{8\pi}{3} \]

Volume

\[ \text{Vol}(\Omega) = \frac{1}{3} \iint_{\partial \Omega} \vec{R} \cdot \vec{N} dA \] where \(\vec{R}(p) = p\) is the radial vector field.

  • \(\text{Vol}(\mathbb{B}^3) = \tfrac{4\pi}{3}\)

4. Source Free Vector Fields

Source Free Vector Fields

Source Free Vector Fields

A vector field \(\vec{F}\) is called source free if \(\divg \vec{F} = 0\).

Other names are solenoidal and incompressible

Stream Functions

A 2d vector field \(\vec{F}\) has a stream function if there is a function \(g\) such that \[ \vec{F} = (\partial_y g, -\partial_x g) = R_{-\pi/2} \left(\nabla g\right) \]

\(\vec{F}\) is tangent to the level curves of \(g\)

Source Free Vector Fields

The following are equivalent

  1. \(\vec{F}\) is source free
  2. The flux across any closed surface is \(0\)
  3. 2d simply connected: \(\vec{F}\) had a stream function \(g\)

\(\vec{F} = (-y, x)\) has stream function \(g = \frac{x^2}{2} + \frac{y^2}{2}\)
