2. FTC for Gradients

FTC for Gradients

FTC for Gradients

\[ \int_{C} \nabla f \cdot \vec{T} ds = f(\vecna{c}(b)) - f(\vecna{c}(a)) \]

Example

  • \(\vecna{c}(t) = \big(t^2/4, \sin^3(t\pi/2)\big)\) \(t \in [0, 1]\)
  • \(f(x, y) = xy\)

\[ \int_{C} ydx + xdy = \frac{1}{4} \]

Conservative Vector Fields

A vector field is conservative if the work done around a closed loop is zero.

Conservative Vector Fields

The following conditions are equivalent:

  1. \(F\) is conservative
  2. \(F = \nabla f\) for some scalar field \(f\)
  3. The work done by \(F\) along a path joining points \(p\) and \(q\) is independent of the path taken

Inverse square law

\[ \vec{F}(x, y) = \frac{1}{(x^2 + y^2)^{3/2}} (x, y), \quad (x, y) \ne (0, 0) \]

\[ \vec{F} = \nabla \left(\frac{-1}{\sqrt{x^2 + y^2}}\right) \]

Test for Gradients

Test for Gradients

Example

\[ F = (2x e^{x^2 - y}, - e^{x^2 - y}) \]

Example

4. Simply Connected Domains

Simply Connected Domains

Simply Connected Domains

A simply connected domain is a connected open set with no holes.

  • Disc - simply connected
  • Annulus - not simply connected

Vector Fields on Simply Connected Domains

Let \(F = (P, Q)\) be a vector field on a simply connected domain. Then \(F\) is a gradient field if and only \(\partial_y P = \partial_x Q\).

Example

\[ F = (P, Q) = \frac{1}{x^2 + y^2} \left(-y, x\right), \quad (x, y) \neq (0, 0) \]

  • \(\partial_y P = \partial_x Q\)
  • Not a gradient

5. Potential Functions

Potential Functions

Determining Potential Functions

  • \(\partial_y P = \partial_x Q\)
  • Solve \(\nabla f= (\partial_x f, \partial_y f) = (P, Q)\)
  • \(\partial_x f = P \Rightarrow f = \int P dx + h(y)\)
  • Sub into \(\partial_y f = Q\) and solve for \(h\)

Example

\[ F(x, y) = (2xy, x^2 + e^y) \]

Example

\[ F(x, y) = (2xy, x^2 + x e^y) \]
