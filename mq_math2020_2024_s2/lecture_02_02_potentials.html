2. Test for Gradients

Test for Gradients

Test for Gradients

\(F = (P, Q) = \nabla f = (\partial_x f, \partial_y f)\) satisfies \[ \partial_y P = \partial_x Q \]

Example

\[ F = (2x e^{x^2 - y}, - e^{x^2 - y}) \]

Example

\[ F = (\cos y, x^2) \]

3. Simply Connected Domains

Simply Connected Domains

Simply Connected Domains

A simply connected domain is a connected open set with no holes.

  • Disc - simply connected
  • Annulus - not simply connected

Vector Fields on Simply Connected Domains

Let \(F = (P, Q)\) be a vector field on a simply connected domain. Then \(F\) is a gradient field if and only \(\partial_y P = \partial_x Q\).

Example

\[ F = (P, Q) = \frac{1}{x^2 + y^2} \left(-y, x\right), \quad (x, y) \neq (0, 0) \]

  • \(\partial_y P = \partial_x Q\)
  • Not a gradient

4. Potential Functions

Potential Functions

Determining Potential Functions

  • \(\partial_y P = \partial_x Q\)
  • Solve \(\nabla f= (\partial_x f, \partial_y f) = (P, Q)\)
  • \(\partial_x f = P \Rightarrow f = \int P dx + h(y)\)
  • \(\partial_y f = Q \Rightarrow \partial_y \int P dx + \partial_y h = Q\)

Example

\[ F(x, y) = (2xy, x^2 + e^y) \]

Example

\[ F(x, y) = (2xy, x^2 + x e^y) \]
