2. Regular curves

Regular Curves

Regular Curves

A \(C^1\) regular, parametrised curve is a \(C^1\) function \[ c(t) = (x_1(t), \dots, x_n(t)) \] with \(c'(t) \neq 0\) and \(t \in (a, b)\).

Examples

  • Circle: \(\vec{c}(t) = (\cos(t), \sin(t))\), \(0 \leq t \leq 2\pi\)
  • Helix: \(\vec{c}(t) = (\cos(t), \sin(t), t)\), \(t \in \mathbb{R}\)
  • Parabola: \(\vec{c}(t) = (t, t^2)\), \(t \in \mathbb{R}\)
  • Cardiod: \(\vec{c}(t) = \big((1-\cos(t))\cos(t), (1-\cos(t))\sin(t)\big)\)

3. Arc Length

Arc Length

Arc Length

The length, \(L\) (or arc length) of a \(C^1\) regular, parametrised curve \({c : [a, b] \to \RR^n}\) is \[ L = \int_C ds = \int_a^b \abs{c'(t)} dt \] where \(ds = \abs{c'(t)} dt\).

Arc Length Motivation

Partition \([a, b]\) as \(a = t_0 < t_1 \cdots < t_{n-1} < t_n = b\).

\begin{align*} L &\simeq \sum_{i=1}^n \abs{c(t_i) - c(t_{i-1})} \\ &= \sum_{i=1}^n \abs{\frac{c(t_i) - c(t_{i-1})}{t_i - t_{i-1}}} (t_i - t_{i-1}) \\ &= \sum_{i=1}^n \abs{\frac{\Delta c}{\Delta t}} \Delta t \to \int_a^b \abs{c'(t)} dt \text{ as } n \to \infty. \end{align*}

Arc Length Examples

  • Straight line: \(c(t) = p + t V\)
  • Circle: \(c(t) = (R \cos t, R \sin t)\)
  • Parabola: \(c(t) = (t, t^2)\)

4. Scalar Line Integrals

Scalar Line Integrals

Scalar Line Integrals

The integral of \(f\) along \(C\) is \[ \int_C f ds = \int_a^b f(c(t)) \abs{c'(t)} dt \]

Scalar Line Examples

\[ c(t) = (\cos t, \sin t, t) \] \[ f(x, y, z) = x^2 + y^2 + z^2. \]
