2. Regular curves

Regular Curves

Regular Curves

A \(C^1\) regular, parametrised curve is a \(C^1\) function \[ \vecna{c}(t) = (x_1(t), \dots, x_n(t)) \] with \(\vecna{c}'(t) \neq 0\) and \(t \in (a, b)\).

Examples

  • Circle: \(\vecna{c}(t) = (\cos(t), \sin(t))\), \(0 \leq t \leq 2\pi\)
  • Helix: \(\vecna{c}(t) = (\cos(t), \sin(t), t)\), \(t \in \mathbb{R}\)
  • Parabola: \(\vecna{c}(t) = (t, t^2)\), \(t \in \mathbb{R}\)
  • Cardiod: \(\vecna{c}(t) = \big((1-\cos(t))\cos(t), (1-\cos(t))\sin(t)\big)\)

3. Tangent Line

Tangent Line

Velocity and Unit Tangent Vector

  • Velocity vector: \(\vecna{c}'\)
  • Unit tangent vector: \(\frac{\vecna{c}'}{\abs{\vecna{c}'}}\)

Tangent Line

The tangent line is the line \(L\) through \(p = \vecna{c}(t)\) in direction \(\vec{V} = \vec{T}(t)\)

\[ L(u) = p + u \vec{V} \]

Example

\[ \vecna{c}(t) = (\cos t, \sin t), \quad t = 0 \]

\[ L(u) = (1, 0) + u(0, 1) = (1, u) \]

4. Arc Length

Arc Length

Arc Length

The length, \(\arclen\) (or arc length) of a \(C^1\) regular, parametrised curve \({\vecna{c} : [a, b] \to \RR^n}\) is \[ \arclen = \int_C ds = \int_a^b \abs{\vecna{c}'(t)} dt \] where \(ds = \abs{\vecna{c'}(t)} dt\).

Arc Length Motivation

Partition \([a, b]\) as \(a = t_0 < t_1 \cdots < t_{n-1} < t_n = b\).

\begin{align*} \arclen &\simeq \sum_{i=1}^n \abs{\vecna{c}(t_i) - \vecna{c}(t_{i-1})} \\ &= \sum_{i=1}^n \abs{\frac{\vecna{c}(t_i) - \vecna{c}(t_{i-1})}{t_i - t_{i-1}}} (t_i - t_{i-1}) \\ &= \sum_{i=1}^n \abs{\frac{\Delta \vecna{c}}{\Delta t}} \Delta t \to \int_a^b \abs{\vecna{c}'(t)} dt \text{ as } n \to \infty. \end{align*}

Arc Length Examples

  • Straight line: \(\vecna{c}(t) = p + t V\)
  • Circle: \(\vecna{c}(t) = (R \cos t, R \sin t)\)
  • Parabola: \(\vecna{c}(t) = (t, t^2)\)
