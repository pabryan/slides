1. Fundamental Theorem of Calculus

Fundamental Theorem of Calculus

Fundamental Theorem of Calculus

Let \(F\) be a differentiable function with \(F'\) continuous. Then \[ \int_a^b F'(x) dx = F|_a^b = F(b) - F(a) \]

Example

\[ \int_2^5 2 x dx \]

Example

\[ \int_2^5 x dx \]

Example

\[ \int_0^{\pi} \sin(x) dx \]

2. Anti-derivatives

Anti-derivatives

Anti-Derivative

Given a function \(f\), any other function \(F\) such that \(F' = f\) is called an anti-derivative for \(f\).

Fundamental Theorem of Calculus

The function \[ F(x) = \int_a^x f(t) dt \] is an anti-derivative for \(f\).

Indefinite Integral

The indefinite integral of \(f\) is any anti-derivative and is written \(\int f(x) dx\).

Example

\(f(x) = 2x\)

3. Common Integrals

Common Integrals

Common Integrals

  • \(\int x^n dx = \frac{x^{n+1}}{n+1} + C\)
  • \(\int \sin x dx = -\cos x + C\)
  • \(\int \cos x dx = \sin x + C\)
  • \(\int e^x dx = e^x + C\)
