1. Linearity

Linearity

Linearity

The derivative is linear: \[ (Af + Bg)' = Af' + Bg'. \]

Example

Compute the derivative of \[ f(x) = 5x^2 + 3x. \]

2. Product Rule

Product Rule

Product Rule

\[ (fg)' = fg' + f'g. \]

Example

Show that \(\tfrac{d}{dx} x^2 = 2x\).

Example

Show that \(\tfrac{d}{dx} x^3 = 3x^2\).

3. Quotient Rule

Quotient Rule

Quotient Rule

At points where \(g \neq 0\), \[ \left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2} \]

Example

Compute the derivative of \(\frac{1}{x}\).

Example

Compute the derivative of \(\tfrac{x^2 + 1}{x - 2}\) for \(x \neq 2\).

4. Trig Functions

Trig Functions

Trig Functions

  • \(\frac{d}{dx} \sin x = \cos x\)
  • \(\frac{d}{dx} \cos x = -\sin x\)
  • \(\frac{d}{dx} \tan x = (\sec x)^2\)

5. Exponential Function

Exponential Function

Exponential Function

\[ \frac{d}{dx} e^x = e^x \]
