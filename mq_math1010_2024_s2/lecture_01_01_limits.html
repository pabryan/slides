1. Limits

Limits

Limits

If the function values \(f(x)\) approach \(L\) as the values \(x\) approach \(a\), then the limit exists and we write \[ \lim_{x\to a} f(x) = L. \] Note: Here we let \(x\) approach \(a\) but we consider only \(x \neq a\).

Sum Law

If the limits \(\lim_{x\to a}f(x)\) and \(\lim_{x\to a} g(x)\) exist, then

\[ \lim_{x \to a} \left[f(x) + g(x)\right] = \lim_{x\to a} f(x) + \lim_{x\to a} g(x) \]

Product Law

If the limits \(\lim_{x\to a}f(x)\) and \(\lim_{x\to a} g(x)\) exist, then

\[ \lim_{x \to a} \left[f(x) g(x)\right] = \left[\lim_{x\to a} f(x)\right] \left[\lim_{x\to a} g(x)\right] \]

Quotient Law

If the limits \(\lim_{x\to a}f(x)\) and \(\lim_{x\to a} g(x)\) exist and if \(\lim_{x\to a} g(x) \neq 0\), then

\[ \lim_{x \to a} \frac{f(x)}{g(x)} = \frac{\displaystyle\lim_{x\to a} f(x)}{\displaystyle\lim_{x\to a} g(x)} \]

Example

Calculate the limit, \[ \lim_{x\to 3} 2 x^2 + 5x - 7 \]

Example

Calculate the limit, \[ \lim_{x\to 2} \frac{x^2 - 4}{x-2} \]

2. One Sided Limits

One Sided Limits

One Sided Limits (Left)

If the function values \(f(x)\) approach \(L\) as the values \(x\) approach \(a\) from the left, then the limit from the left exists and we write \[ {\color{red} \lim_{x\to a^-}} f(x) = L. \] Note: To say that \(x\) approaches \(a\) from the left means that we restrict to \({\color{red} x < a}\).

One Sided Limits (Right)

If the function values \(f(x)\) approach \(L\) as the values \(x\) approach \(a\) from the right, then the limit from the right exists and we write \[ {\color{red} \lim_{x\to a^+}} f(x) = L. \] Note: To say that \(x\) approaches \(a\) from the right means that we restrict to \({\color{red} x > a}\).

Limits and One Side Limits

\[ \lim_{x\to a} f(x) = L \] if and only if \[ \lim_{x\to a^-} f(x) = L \text{ and } \lim_{x\to a^+} f(x) = L \]

Example

Calculate the left and right limits of the function \[ f(x) = \begin{cases} x + 1, & x \leq 2 \\ x^2, & x > 2 \end{cases} \] as \(x \to 2\).

3. Infinite Limits

Infinite Limits

Positive Infinite Limits

If the functions values \(f(x)\) become positive and unbounded as \(x \to a\), then we write \[ \lim_{x\to a} f(x) = \infty. \]

Negative Infinite Limits

If the functions values \(f(x)\) become negative and unbounded as \(x \to a\), then we write \[ \lim_{x\to a} f(x) = -\infty. \]

Example

Calculate the limit \[ \lim_{x\to 0} \frac{1}{x^2} \]

Example

Calculate the limit \[ \lim_{x \to 1} \frac{x+1}{x-1} \]

4. Squeeze Theorem

Squeeze Theorem

Squeeze Theorem

Suppose that \(\textcolor{green}{f(x)} \leq \textcolor{orange}{g(x)} \leq \textcolor{green}{h(x)}\) and that \[ \lim_{x \to a} \textcolor{green}{f(x)} = \lim_{x \to a} \textcolor{green}{h(x)} = L. \] Then \[ \lim_{x\to a} \textcolor{orange}{g(x)} = L. \]

Example

Evaluate the limit

\[ \lim_{x\to 0} x^2 \sin \tfrac{1}{x}. \]

Example

Evaluate the limit \[ \lim_{\theta \to 0} \sin \theta \]
