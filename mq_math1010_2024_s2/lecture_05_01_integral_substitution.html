2. Reversing the Chain Rule

Reversing the Chain Rule

Chain Rule

\(H(x) = F(g(x))\) is an anti-derivative for \(f(g(x)) g'(x)\) where \(F' = f\)

Example

Let \(f(u) = \cos(u)\) and \(g(x) = x^2\)

Substitution

If \(F' = f\) \[ \int f(g(x) g'(x) dx = F(g(x)) + C \]

Example

Calculate \[ \int (x-4)^2 dx \]

Example

Calculate \[ \int x \sqrt{x^2 - 5} dx \]
