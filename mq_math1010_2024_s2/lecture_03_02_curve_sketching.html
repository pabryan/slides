1. Increasing and Decreasing Functions

Increasing and Decreasing Functions

Increasing and Decreasing Functions

A function \(f\) is said to be increasing if \(f(x_2) \geq f(x_1)\) whenever \(x_2 \geq x_1\).

A function \(f\) is said to be decreasing if \(f(x_2) \leq f(x_1)\) whenever \(x_2 \geq x_1\).

Example

\(f(x) = x^2\)

First Derivative

Let \(f\) be a differentiable function.

  • Increasing \(f' \geq 0\)
  • Decreasing: \(f' \leq 0\)

Example

\(f(x) = \cos x\).

2. Extreme Points

Minimum and Maximum

Minimum and Maximum

  • Minimum: \(f(x) \geq f(x_{\min})\)
  • Maximum: \(f(x) \leq f(x_{\max})\)

Extreme Value Theorem

A continuous function defined on a closed, bounded interval \([a, b]\) attains both a maximum and minimum.

Example

\(f(x) = x^2 + 1\) for \(x \in [-2, 1]\).

Local Min and Max

A function \(f\) has a local minimum at \(x_0\) if \(f(x) \geq f(x_0)\) for every \(x\) in some open interval containing \(x_0\).

A function \(f\) has a local maximum at \(x_0\) if \(f(x) \leq f(x_0)\) for every \(x\) in some open interval containing \(x_0\).

Example

\(f(x) = x^3 - x = x(x-1)(x+1)\).

3. Critical Points

Critical Points

Critical Points

A critical point for a function \(f\) is point \(x\) where \(f'(x) = 0\) or \(f'(x)\) is not defined.

Example

Let \(f(x) = x^3 - x = x(x-1)(x+1)\).

First Derivative Test

If a function \(f\) has a local minimum or maximum at the point \(x\), then \(x\) is a critical point.

Example

\(f(x) = x^3 - x\)

Example

Let \(f(x) = |x|\).

4. Concavity

Concavity

Second Derivative

Let \(f\) be a differentiable function. If \(f'\) is also differentiable, we say that \(f\) is twice differentiable and write \(f''\) for the derivative of \(f'\).

Second Derivative Test

  • local minimum: \(f'' \geq 0\)
  • local maximum: \(f'' \leq 0\)

Example

\(f(x) = x^2\)

Example

\(f(x) = x^4\)

Example

\(f(x) = x^3\)

5. Asymptotes

Asymptotes

Vertical Asymptote

Vertical asymptote: \(\lim_{x\to x_0^{\pm}} = \pm \infty\)

Example

\(f(x) = \frac{1}{x^2}\)

Example

\(f(x) = \frac{1}{x-1}\)

Limits at Infinity

If \(f(x)\) approaches \(L\) as \(x\) becomes arbitrarily large we write \[ \lim_{x\to \infty} f(x) = L. \] Similarly for \(\lim_{x \to -\infty} f(x) = L\).

Example

\[ \lim_{x \to \infty} x^2 = \infty \]

Example

\(f(x) = \tfrac{x}{x+1}\)

Example

\(f(x) = \tfrac{x}{x^2+1}\)

Horizontal Asymptote

If \(\lim_{x\to\pm \infty} f(x) = L\) then \(f\) has a horizontal asymptote \(L\) at \(\pm \infty\).

Example

\(f(x) = \tfrac{x}{x+1}\)
